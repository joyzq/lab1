# 2D Игра с графической библиотекой

Проект по созданию небольшой 2D-игры с использованием библиотеки Pygame.

## Описание

Этот проект представляет собой 2D-игру, в которой игрок может перемещаться по карте, собирать предметы и достигать выхода.

Чтобы начать игру запустите файл game.py 

## Управление

- **W** - движение вверх 
- **S** - движение вниз 
- **A** - движение влево 
- **D** - движение вправо 
- **ESC** - выход из игры

## Правила игры

1. Соберите все коллекционные предметы
2. Достигните выхода после сбора всех предметов
3. Избегайте патрулей в бонусной версии
4. Минимизируйте количество движений

## Соответствие стандартам

Проект соответствует стандарту flake8. Для проверки используйте:

flake8 game.py
